Instruções de Uso

QUESTÃO 1:
Abra o Octave: utilize a versão octave 9.2

Navegue até o diretório onde o arquivo está localizado e execute o programa.

Digite os Dados de Entrada:

O script solicitará que você insira o número de vetores de entrada.
Para cada vetor, insira os valores das características x1 e x2 exemplo, [1, 2] (obs. utilize os colchetes).
Para cada vetor, insira o rótulo correspondente, que deve ser 0 ou 1.

Resultados:

Após o treinamento, o script exibirá a porcentagem de acerto final do modelo.
A porcentagem de acerto indica a proporção de previsões corretas feitas pelo Perceptron.
